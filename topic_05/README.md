# Тема 5. Основи технології Docker

Базовий приклад контейнеризації Node.js-застосунку: Dockerfile, `.dockerignore`, запуск `docker compose`.

## Структура теми

- [`src`](./src/) - простий Express-додаток для контейнерного запуску.
- [`Dockerfile`](./Dockerfile) - інструкції зборки image.
- [`docker-compose.yml`](./docker-compose.yml) - локальний запуск контейнера сервісу.
- [`.dockerignore`](./.dockerignore) - виключення файлів із build context.

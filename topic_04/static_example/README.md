# static_example

Демо віддачі статичних файлів у Express із фронтенд-ресурсами в `public`.

## Файли прикладів

- [`app.ts`](./app.ts) - Express-сервер із налаштуванням статичної директорії.
- [`public/index.html`](./public/index.html) - сторінка для перевірки віддачі статичного контенту.
- [`public/css/styles.css`](./public/css/styles.css) - стилі сторінки.
- [`public/js/script.js`](./public/js/script.js) - клієнтський JS для прикладу.

# upload_example

Приклади завантаження файлів через `multer`: простий upload, налаштування збереження та multi-upload.

## Файли прикладів

- [`app-simple.ts`](./app-simple.ts) - мінімальний варіант single-file upload.
- [`app.ts`](./app.ts) - основний приклад upload-обробки.
- [`app-diskstorage.ts`](./app-diskstorage.ts) - кастомне збереження файлів через `diskStorage`.
- [`public/upload.html`](./public/upload.html) - форма для одного файлу.
- [`public/multi.html`](./public/multi.html) - форма для кількох файлів.
- [`public/docs.html`](./public/docs.html) - коротка довідкова сторінка по сценаріях upload.

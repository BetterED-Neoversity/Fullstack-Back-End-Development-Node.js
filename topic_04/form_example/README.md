# form_example

Приклади роботи з формами в Express: сторінки форм, прийом даних і базова обробка запитів.

## Файли прикладів

- [`app.ts`](./app.ts) - сервер із маршрутами для формових сценаріїв.
- [`public/index.html`](./public/index.html) - стартова сторінка з переходами на форми.
- [`public/search.html`](./public/search.html) - приклад пошукової форми.
- [`public/register.html`](./public/register.html) - приклад форми реєстрації.

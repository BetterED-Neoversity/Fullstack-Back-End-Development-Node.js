# Тема 4. Статичні файли, обробка форм та завантаження файлів в Express

Навчальні приклади з `Express` для SSR-шаблонів, віддачі статичних ресурсів, роботи з HTML-формами та upload через `multer`.

## Структура теми

- [`static_example`](./static_example/) - підключення `express.static`, HTML/CSS/JS.
  Деталі: [`static_example/README.md`](./static_example/README.md)
- [`template_example`](./template_example/) - серверний рендеринг через `EJS`.
  Деталі: [`template_example/README.md`](./template_example/README.md)
- [`form_example`](./form_example/) - обробка GET/POST формових запитів.
  Деталі: [`form_example/README.md`](./form_example/README.md)
- [`upload_example`](./upload_example/) - single/multiple upload файлів через `multer`.
  Деталі: [`upload_example/README.md`](./upload_example/README.md)

## Швидкий старт

```bash
npm run start:static
npm run start:ejs
npm run start:form
npm run start:upload
```

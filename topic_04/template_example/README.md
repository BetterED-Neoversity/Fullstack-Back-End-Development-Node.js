# template_example

Приклади шаблонізації через `EJS`: рендер сторінок, передача даних у view та використання partials.

## Файли прикладів

- [`app.ts`](./app.ts) - коректний приклад налаштування `EJS` і рендеру сторінок.
- [`app_wrong.ts`](./app_wrong.ts) - приклад із типовою помилкою конфігурації для розбору.
- [`views/index.ejs`](./views/index.ejs) - головна сторінка.
- [`views/about.ejs`](./views/about.ejs) - сторінка "Про нас".
- [`views/books.ejs`](./views/books.ejs) - приклад рендеру даних списку.
- [`views/partials/header.ejs`](./views/partials/header.ejs) - спільна шапка.
- [`views/partials/navbar.ejs`](./views/partials/navbar.ejs) - навігація.
- [`views/partials/footer.ejs`](./views/partials/footer.ejs) - спільний футер.
